# Thresholding

##### Description

Return a pass/fail value based on a thresholding value, per cell

##### Usage

Input projection|.
---|---
`y-axis - Layer 1`        | numeric, value to flag
`y-axis - Layer 2`        | numeric, threshold value

Input parameters|.
---|---
`operand`        | operand for comparison of values to threshold

Output relations|.
---|---
`flag`        | pass / fail flag
